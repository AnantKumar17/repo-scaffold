## Refactoring Checklist

### Before Refactoring
- [ ] Identify all packages affected
- [ ] Check inter-package dependencies
- [ ] Run all tests to establish baseline
- [ ] Create a branch for the refactor

### During Refactoring
- [ ] Update imports in all affected packages
- [ ] Update package.json dependencies if moving code
- [ ] Maintain API compatibility or update all callers
- [ ] Keep commits focused and atomic

### After Refactoring
- [ ] Run tests in all affected packages
- [ ] Run build in dependency order
- [ ] Check for circular dependencies
- [ ] Update documentation
- [ ] Test integration between packages
